# Changelog

All notable changes to this package will be documented in this file.

## [0.0.3] - 2025-11-11
* Initial release of Flutter NumKit.
* Added extensions for:
    - Angle (degToRad, radToDeg)
    - Boolean (isEvenNum, isOddNum, isBetween)
    - Duration (sec, min, milSec, micSec, hours, days)
    - EdgeInsets (allEdgeInsets, verticalEdgeInsets, horizontalEdgeInsets, etc.)
    - Gap (vGap, hGap, squareGap, spacer)
    - Percentage (percent)
    - Radius (allRadius, allBorderRadius, verticalBorderRadius, horizontalBorderRadius, onlyBorderRadius)
    - ScreenSize (ofWidth, ofHeight)
    - Tween (lerp)
* Fully documented with usage examples.
